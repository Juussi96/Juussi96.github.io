#Synerra

#Projketin taustat ja tavoitteet
Projektin tavoitteena oli suunnitella ja toteuttaa selainpohjainen palvelu, joka helpottaa kilpai-lullisten videopelien pelaajia löytämään sopivaa peliseuraa. Palvelun ydinidea on tarjota käyttä-jille mahdollisuus etsiä pelikavereita erilaisten filttereiden perusteella sekä mahdollistaa ”syvem-pi” tutustuminen reaaliaikaisen chatin avulla.
Synerra on suunnattu kaikille kilpailullisten pelien harrastajille. Se palvelee sekä uusia pelaajia, jotka etsivät turvallista ympäristöä kehittymiselle, että myös kokeneita pelaajia, jotka kaipaavat tavoitteellisempaa pelaamista. Kokonaisuutena projektin visiona oli luoda moderni, helppokäyt-töinen ja yhteisöllinen alusta, joka vähentää toksisuutta ja tekee pelikavereiden löytämisestä sujuvaa ja miellyttävää.
